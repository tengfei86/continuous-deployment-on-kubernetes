= Delete Custom Attributes

You can delete any custom attribute by providing business object name and attribute name.

== 1. Request URL

/{PDM Server domain}/metadata/write/deleteCustomAttributes

*Methodï¼š* POST

== 2. Request headers

[cols=",",options="header",]
|===
|Name |Value
|Content-Type |application/json
|===

== 3. Response Type

[cols=",",options="header",]
|===
|Name |Value
|responseType |blob
|===

== 4. Request body

[source,json]
----
{
    "BUSINESS OBJECT ATTR": {
        "language": "en",
        "readBack": false,
        "timezone": "GMT+08:00",
        "showSQLStats": true,
        "data": [
            {
                "BO_NAME": {{BO_NAME1}},
                "BO_ATTR_NAME": {{BO_ATTR_NAME1}}
            },
            {
                "BO_NAME": {{BO_NAME2}},
                "BO_ATTR_NAME": {{BO_ATTR_NAME2}}
            }
            ...
        ]
    }
}
----
